# Complete list of IT companies addresses and contacts

This list was created for the purpose of helping people interested in being hired in IT industry.

## Data collection

All information here is gathered via Google Maps API using [SerpAPI](https://serpapi.com/) and is free to use.

The search queries used: `tech`.

The queries were used for a list of town in each of the countries.
